# NetRecon Scanner

![NetRecon Scanner](/project-scanner.png)

Herramienta de reconocimiento de red y escáner de vulnerabilidades personalizado inspirado en Nmap, desarrollado completamente en Python utilizando módulos como `Scapy` para manipulación de paquetes.

## Capacidades Principales
- **Escaneo de Puertos**: Escaneos TCP SYN y UDP multihilo.
- **Enumeración de Servicios**: Banner grabbing y fingerprinting de versiones.
- **Detección Automatizada de Vulnerabilidades**: Referencias cruzadas de servicios descubiertos con una base de datos local de CVEs.
- **Reportes Exportables**: Genera resúmenes profesionales en JSON y Markdown de la postura de red.
