# CH4R0N IV: Portfolio 2026

![Portfolio](/project-portfolio.png)

Este proyecto representa el desarrollo de este mismo sitio web. Diseñado desde cero para reflejar una estética oscura, profesional y cyberpunk inspirada en temas como *Tetsuo*.

## Arquitectura
- **Frontend**: Construido con `React` y empaquetado con `Vite` para máximo rendimiento.
- **Estilos**: Variables CSS puras y utilidades personalizadas para máxima flexibilidad sin depender de frameworks pesados.
- **Gestión de Contenido**: Un motor de renderizado Markdown a medida que permite desplegar nuevos posts de blog, writeups de CTF y casos de estudio de proyectos simplemente colocando archivos `.md` en el directorio `src/posts/`.
